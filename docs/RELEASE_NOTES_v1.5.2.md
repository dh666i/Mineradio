# Mineradio v1.5.2

本版本集中修复多显示器兼容和音乐库面板布局问题。

## 多显示器与窗口恢复

- 主窗口现在可以正常拖动到副屏，不会在跨屏过程中被强制拉回原显示器。
- 登录窗口同样支持跨显示器移动。
- 退出后会保存主窗口所在显示器、位置、尺寸和最大化状态，下次启动自动恢复。
- 支持位于主屏左侧或上方的副屏、负坐标以及不同 DPI 缩放比例。
- 显示器断开、重新排列或分辨率变化后，会自动把不可操作的窗口恢复到可见区域。

## 音乐库界面修复

- 播放队列中的歌曲名与歌手恢复为独立两行。
- 长歌曲名和歌手信息自动省略，不再撑出横向滚动条。
- 行级操作按钮仅在悬停或键盘聚焦时显示，并为正文保留空间。
- 歌单展开后直接复用原歌单卡片作为标题，移除重复封面、标题和曲目数。
- 移除歌单详情多余的嵌套玻璃背景，操作栏改为“播放歌单”和收起按钮。

## 依赖安全维护

- 更新网络请求与请求解析依赖，修复已公开的拒绝服务和原型污染风险。
- 同步更新构建链路中的归档与配置解析依赖安全补丁。

## 安装提示

- 本版本继续使用自 `v1.3.0` 起沿用的同一份自签名 Authenticode 证书。
- 自签名证书不能建立 SmartScreen 公共信誉，Windows 仍可能显示未知发布者提示。
- 只从 `dh666i/Mineradio` Releases 下载，并核对 Release 提供的 SHA256。
- 从旧版本覆盖安装时，播放设置、登录状态、队列和本地用户资源应保留。
