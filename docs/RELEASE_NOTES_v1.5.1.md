# Mineradio v1.5.1

## 本次更新

- 沉浸模式改为直接进入应用全屏，界面元素默认隐藏，鼠标靠近对应屏幕边缘时显示搜索、账号区、视觉控制台和完整播放器控制栏。
- 移除简洁迷你播放器样式，普通模式与沉浸模式统一保留原版完整控制栏。
- 重做 Electron 全屏状态同步与窗口恢复，修复 Esc、窗口按钮和沉浸模式切换时的状态反复、退出不完整和窗口尺寸丢失问题。
- 启动时优先恢复上次歌曲、队列和播放位置；没有可恢复歌曲时默认进入 Home，不再停留在空白播放器页面。
- 优化右上角窗口控制区的边距，使按钮不再紧贴窗口边缘。
- 修复搜索页横向溢出、窄窗口压缩异常、更多操作省略号裁切、输入框焦点边框和结果计数文案。
- 打开设置、登录等弹窗时会先退出沉浸模式，避免弹层被隐藏。

## 安装提示

- 本版本继续使用自 `v1.3.0` 起沿用的同一份自签名 Authenticode 证书。
- 自签名证书不能建立 SmartScreen 公共信誉，Windows 仍可能显示未知发布者提示。
- 只从 `dh666i/Mineradio` Releases 下载，并核对 Release 提供的 SHA256。
- 从旧版本覆盖安装时，播放设置、登录状态、队列和本地用户资源应保留。
