# Mineradio v1.4.0

## 本次更新

- Home 以每日推荐为主入口，并补齐我的歌单、红心歌曲、继续听、播放队列和推荐内容。
- 网易云与 QQ 音乐搜索支持分页和加载更多；单个平台暂时不可用时保留另一平台结果并可重试。
- 新增专辑详情、歌手专辑，以及搜索结果的播放全部、下一首、收藏和详情快捷操作。
- 播放队列可在下次启动时恢复，支持拖动/键盘排序、多选移除、撤销和保存为网易云歌单。
- 本人创建的普通网易云歌单支持重命名、删除、移除歌曲和曲序同步。
- 歌词支持翻译与罗马音副行，网易云 YRC 和 QQ QRC 逐字高亮保持有效；桌面歌词同步显示副行。
- 播客新增前后 15 秒、倍速和睡眠定时。
- 新增设置中心，并补齐弹窗焦点、键盘进度控制和减少动态选项。
- 更新下载显示线路、速度与预计时间，并可在下载过程中真正取消。
- 官方网页登录状态改为主进程落盘并向界面返回脱敏结果；登录过期、网络不可用和资料暂不可用不再混为同一种状态。

## 安装提示

- 本版本继续使用与 `v1.3.0` 至 `v1.3.2` 相同的自签名 Authenticode 证书。
- 自签名证书不能建立 SmartScreen 公共信誉，Windows 仍可能显示未知发布者提示。
- 只从 `dh666i/Mineradio` Releases 下载，并核对 Release 提供的 SHA256。
- 从旧版本覆盖安装时，播放设置、登录状态和本地用户资源应保留；首次启动会按新队列结构保存后续播放状态。
