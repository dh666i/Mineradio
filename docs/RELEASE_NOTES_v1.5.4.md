# Mineradio v1.5.4

本版本新增网易云顺序播放的无缝切歌与可选交叉淡化，并集中修复搜索、歌单、启动恢复和临时面板交互。

## 无缝切歌与交叉淡化

- 音频设置新增“关闭 / 无缝衔接 / 交叉淡化 3 秒 / 5 秒 / 8 秒”。
- 默认使用无缝衔接；顺序播放会在当前歌曲结束前提前准备下一首网易云音源。
- 交叉淡化使用双音频槽和等功率曲线，两个音频槽共享频谱、节拍分析、均衡器与主音量输出。
- 设置保存在现有音频配置中，旧版本设置无需手动迁移。

## 播放稳定性

- 候选音源加载失败或启动超时时不会中断当前歌曲，并在短时间内避免重复请求同一失败音源。
- 队列重排后使用稳定歌曲标识重新定位，避免回退或切换到错误歌曲。
- 暂停、手动切歌、拖动进度、清空队列、睡眠定时、切换设置和退出程序都会正确终止双音频过渡。
- 修复旧请求或旧播放 Promise 延迟返回时可能取消、暂停新播放任务的问题。
- 播客、本地音乐、QQ 音乐、私人 FM、单曲循环、随机播放、倍速和“播完本首停止”自动使用普通切歌。
- Windows 媒体控制和锁屏播放状态会按实际可听音频处理失败恢复，不会出现界面已暂停但旧音频仍继续播放。

## 界面与交互

- 修复搜索框与桌面标题栏之间的鼠标悬停断层，经过顶部空隙时搜索框不再意外收起。
- 鼠标移出应用、窗口失焦、最小化或隐藏时，搜索等临时模块会自动收起。
- 恢复上次歌曲时立即恢复中央视觉和歌词舞台，不再需要先点击播放。
- 精简歌单展开详情，统一操作区、歌曲列表、滚动条和收起按钮样式。
- 设置中的“更新与数据”调整为“关于”，保留更新、备份和诊断功能，并增加功能与优化建议反馈入口。

## 安装提示

- 本版本继续使用自 `v1.3.0` 起沿用的同一份自签名 Authenticode 证书。
- 自签名证书不能建立 SmartScreen 公共信誉，Windows 仍可能显示未知发布者提示。
- 只从 `dh666i/Mineradio` Releases 下载，并核对 Release 提供的 SHA256。
- 从旧版本覆盖安装时，播放设置、登录状态、队列和本地用户资源应保留。
