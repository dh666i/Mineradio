# Mineradio v1.2.0

这是 `dh666i/Mineradio` 发布的 `v1.2.0` 版本。

## 本次更新

- 首页改为网易云每日推荐，支持完整列表、播放全部、随机播放和指定歌曲开始播放。
- 新增网易云私人 FM 独立队列，支持自动补歌和不喜欢当前歌曲。
- 搜索结果支持播放全部、批量加入队列和随机播放，并优化网易云登录后的结果优先级。
- 首页重新排版：移除每日推荐内嵌曲目表，“接着听”改为全幅封面卡片并适配短窗口滚动。
- 本地服务限制为回环地址并增加请求来源校验；依赖审计已清除已知漏洞。
- 修复卸载后残留应用资源目录的问题，安装包内包含 GPL、来源、隐私和安全说明。

## 安装提示

- 安装包未进行代码签名，Windows SmartScreen 可能显示“未知发布者”。请只从本 Release 下载，并核对 SHA256 文件。
- 当前版本仍沿用 Mineradio 的应用标识和用户数据目录。已有其他 Mineradio 安装时，请先备份用户数据，不要直接覆盖仍需保留的安装。
- 安装器要求使用独立的 `Mineradio` 目录，避免安装或卸载影响同目录中的其他文件。
