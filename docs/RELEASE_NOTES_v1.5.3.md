# Mineradio v1.5.3

本版本集中完善主程序中央歌词在横屏、竖屏和长歌词场景下的布局与渲染体验。

## 自适应歌词布局

- DIY 视觉控制台新增“自动 / 单句 / 三句”歌词布局选项。
- 自动模式在横屏聚焦当前歌词，在竖屏显示上一句、当前句和下一句。
- 当前歌词在横屏最多显示两行，在竖屏最多显示三行。
- 当前歌词较高时会自动弱化或隐藏上下文，避免歌词互相重叠。
- 翻译与罗马音只跟随当前歌词，不会错误附着到上下句。

## 长歌词排版

- 中文按标点和禁则规则平衡换行，减少行首闭合标点和行尾起始标点。
- 英文优先按单词换行，并保护缩写、连字符、重音字符与组合字符。
- emoji、ZWJ 家庭组合和国旗组合不会在字符中间断开。
- 极端长歌词使用线性分段，避免换行计算随文本长度出现明显卡顿。
- 逐字播放进度会连续映射到换行后的每个视觉行。

## 竖屏与渲染稳定性

- 歌词会根据实际相机距离、FOV 和窗口比例动态适配视锥，为呼吸和节拍缩放保留安全空间。
- 修复竖屏长歌词及超长不可断词可能被画布或屏幕边缘裁切的问题。
- 三句模式换句及样式重建会继承上一帧透明度和光效状态，避免短暂闪空。
- 歌词行类型、透明度与逐字进度编码到纹理，避免多行模式占用过多 Shader uniform。
- 可读性纹理与溢光纹理采用较低分辨率生成，降低显存占用和切句压力。

## 设置兼容

- 歌词布局选择会写入本地设置，并随视觉配置存档导入导出。
- 旧版设置未包含该字段时默认使用自动模式，不影响原有视觉参数。

## 安装提示

- 本版本继续使用自 `v1.3.0` 起沿用的同一份自签名 Authenticode 证书。
- 自签名证书不能建立 SmartScreen 公共信誉，Windows 仍可能显示未知发布者提示。
- 只从 `dh666i/Mineradio` Releases 下载，并核对 Release 提供的 SHA256。
- 从旧版本覆盖安装时，播放设置、登录状态、队列和本地用户资源应保留。
