# Mineradio v1.5.0

## 本次更新

- 新增网易云发现页，支持热门歌单、新碟、新歌和歌单分类浏览。
- 网易云搜索增加歌曲、歌手、专辑和歌单类型切换、完整分页与播放全部。
- 歌单详情支持完整曲目载入、收藏与取消收藏；本人歌单可编辑名称、简介、标签和隐私设置。
- 新增 10 段均衡器、音效预设、基础响度与峰值保护、输出设备选择和下一首预取。
- 播放队列最多持久化 5000 首，大型队列支持窗口分页、跨页定位和稳定批量操作。
- 主窗口、桌面歌词和壁纸窗口启用 Electron 沙箱，并收紧登录导航、页面权限和 IPC 来源校验。
- 增加脱敏持久诊断日志、异常记录、自动化源码检查和 Windows CI。
- 修复设置中心宽度、滚动和文字换行问题，统一音乐库工具按钮尺寸与布局。
- 改进图标按钮、旧式点击目标和设置分类的键盘访问与焦点状态。

## 安装提示

- 本版本继续使用自 `v1.3.0` 起沿用的同一份自签名 Authenticode 证书。
- 自签名证书不能建立 SmartScreen 公共信誉，Windows 仍可能显示未知发布者提示。
- 只从 `dh666i/Mineradio` Releases 下载，并核对 Release 提供的 SHA256。
- 从旧版本覆盖安装时，播放设置、登录状态、队列和本地用户资源应保留。
